# PyConverse — Smart & Friendly Python Chatbot

## Overview
**PyConverse** is a simple, smart, and interactive rule-based chatbot built with Python.  
It responds to greetings, answers basic questions about itself, tells jokes, and demonstrates Python fundamentals like loops, conditionals, and string handling.  
This project is ideal for beginners to learn Python programming while building a fun, practical application.

## Features
- Friendly responses to greetings and common questions  
- Tells random jokes to make the conversation fun 😄  
- Answers questions about itself (name, age, job, feelings)  
- Exit commands: `bye`, `exit`, `quit`  
- Beginner-friendly, clean, and readable Python code  

## Technologies Used
- Python 3.x  
- Built-in libraries: `random`, `datetime`  

## How to Run
1. Clone or download the repository.  
2. Open the project folder and run:

```bash
python chatbot.py
3. Start chatting with the bot in your terminal.
4. Type bye to exit the conversation.


Example Conversation
You: hi
Bot: Hi there! How can I help you today? 😊
You: what is your name
Bot: I am a funny rule-based chatbot built with Python by Priyanka.

You: do you like me
